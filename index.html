Mathematical operators

  • +
  • -
  • *
  • /
  • %

Mathematical operators are used in recent times for calculating sizes, changing colors, and other functionalities.

font-size: $base-size * 5;

color: #102030 + #010203;

Equality operators

  • ==
  • !=

Equality operators are used to check if two values are the same.

Numbers, Strings, Colors, Lists, Maps, Calculations, Functions.

@if( $size != 100% ) { margin: auto; }

@if( input[type==text] ) { font-size: $base-font; }

Logical operators

  • and
  • or
  • not
not true; => false
not false; => true

true and true; => true
true and false; => false

true or false; => true
false or false; => false

@if($size > 15 or $size < 25){ font-size:$size }

Comparison operators

  • >
  • >=
  • <
  • <=

Check whether two values are greater or less than one another.

@if($padding <= 20px) {  
    padding: $padding;  
} @else {  
    padding: $padding / 2;  
}

String operators

  • +
  • -
  • /

+ returns a string writing with quotes.

- returns a string writing without quotes. 

"Ali " + "Imanov" return Ali Imanov
sans + serif return sans serif
